ChEMBL benchmarks

Datasets

Datasets include compounds activity records from experiments on 5-HT1A, 5-HT2A, 5-HT2C, 5-HT6, 5-HT7, A1, A2A, D2, H3, M1 splitted into train/test. Each compound is linked to smiles, year of first publication, murcko scaffold and bioactivity. While few versions are available for research purposes, main benchmarks are run against v1, a version with label derived only from records with Ki value reported.

Dataset can be downloaded here or using our helper scripts. Additionally, see Basic_example.ipynb (independent of drgmum) and Extended_example.ipynb for examples how to load datasets and run basic methods.
